# Earthquake Visualization Project

## Overview
The goal of this project was to create an interactive dashboard visualizaing earthquakes around the world. For any given earthquake, analysis is possible, including aftershock predictions, economic impact descriptions, and regional historic data visualization. Additionally, the filtering of earthquakes based on certain conditions is possible.

## Background
There are a few different projects made to visualize earthquakes, most notably the US Geological Survey quake map. I wanted to improve on these projects by combining Euro - Mediterranean Seismological Center with USGS data.

## Technologies Used
This project is built purely in Python, utilizing the Streamlit library. Pandas and Geopy are used for data manipulation and Ploty and Altair are used for mapping and graphing. Additionally, ThreadPoolExecutor is used to improve data retrieval performance.
